Installation
------------

- Install XLIFF Exchange with ``pip``::

    pip install django-modeltranslation-xliff

- Add ``'modeltranslation_xliff'`` to ``INSTALLED_APPS`` in your project's
  :file:`settings.py`::

    INSTALLED_APPS = (
        'modeltranslation',
        'modeltranslation_xliff',
        ...
    )

Usage example see in :doc:`usage` section.
